# AGENTS.md

## 项目定位

这是当前主线项目：最小化端云语音助手 CLI。当前优先目标是稳定跑通语音闭环，再扩展持续唤醒、多音色、角色化与展示能力。

## 开发前置要求

- 开发任何新功能前，先阅读工作区根目录 `AGENTS.md` 和 `CLAUDE.md`。
- 修改代码前，必须先检查 `2025AIoT-master/` 旧项目与本项目现有实现中是否已有可复用逻辑。
- 能复用的旧实现、接口、命令、音频处理经验、错误处理方式，应优先复用或适配。
- 不可忽视旧项目已有实现而直接从零编写同类代码。
- 如果不能复用旧实现，需要简要说明原因。

## 同步要求

修改主线代码后，同步更新：

- `.codex/任务进度面板.md`
- `.codex/开发计划.md`
- 必要时同步 `.codex/当前实现逻辑.md`
